# financial-backend

## Setup

1. Install dependencies
```
pip install -r requirements.txt
```
2. Create a `.env` file in the root directory with the following variables:
```
PINATA_API_KEY=
PINATA_SECRET_KEY=
GEMINI_API_KEY=
```
